# Fauxmentum dashboard
A simple momentum-like Chrome extension I made to practice async javascript.

You can create your own in the **Working with APIs** module of [scrimba.com](www.scrimba.com)'s Frontend Developer path.

To install:
1. Download the code from Github:
Click Code → Download ZIP, then save it somewhere you’ll remember.

2. Unzip the project so you have a normal folder with the extension files inside.

3. Open Chrome’s extensions page by entering `chrome://extensions/`
in the address bar.

4. On the extensions page, toggle *Developer mode* (in the top right corner).

5. Click the  **Load unpacked** button, and then select the unzipped project folder (the one with `manifest.json`).

6. Confirm it’s installed. Make sure it appears in the list. Open a new tab, and your momentum dashboard should open.<br/>
